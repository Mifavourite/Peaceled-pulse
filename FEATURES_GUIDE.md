Track your emotional state and identify triggers.

- **Personal Notes**: Add detailed notes about your day

**How to Use:**
1. Tap "Check-In" on dashboard
2. Select your current mood
3. Identify any triggers you experienced
4. Add optional notes
5. Complete check-in

**Benefits:**
- Build self-awareness of patterns
- Identify trigger trends over time
- Track emotional progress
- Access help when needed

Immediate access to crisis resources and coping strategies.

**Features:**

- Red phone icon in top navigation (always visible)
- "Support" button on dashboard
- From daily check-in when struggling

**Emergency Features:**
- One-tap calling
- Copy numbers to clipboard
- Send pre-composed text messages

- **Detection Logging**: Track when protection helped you
- **Red Screen Warning**: Clear visual feedback

## üîí Security Features

### Encryption & Privacy
- **AES-256 Encryption**: All data encrypted at rest
- **SQLCipher Database**: Hardware-backed encryption
- **Secure Storage**: Platform-specific secure storage (Keychain/Keystore)
- **No Cloud Sync**: All data stays on your device
- **Password Protection**: Bcrypt password hashing (cost factor 12)
- **Biometric Auth**: Fingerprint/Face ID support

### Security Architecture
- **Zero-Trust Design**: Security at every layer
- **Certificate Pinning**: Protected network communications
- **Intrusion Detection**: Real-time security monitoring
- **Secure Key Management**: Hardware-backed key storage
